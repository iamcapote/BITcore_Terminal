Coordinate GitHub research automation from a single surface. Verify repository connectivity, browse the remote research tree, pull individual files for editing, and publish updates after reviewing staged changes. The live activity feed mirrors CLI telemetry so you never miss a commit, warning, or error.

  • Verify validates your stored GitHub credentials, repository permissions, and branch status.
  • List inspects the remote research directory to plan pulls or uploads.
  • Fetch stages the latest remote copy of an artifact for editing.
  • Upload publishes the active staged file, while Push commits every dirty staged change in one batch.
